## Get Invoice

Ambil detail invoice berdasarkan kode booking.

### Path Parameters

<ParamField path="kodeBooking" type="string" required>
  Kode booking (format: `BK2604-XXXX`)
</ParamField>

### Response

<ResponseField name="booking" type="object">
  Data booking dengan info paket, vendor, dan payments
</ResponseField>

<ResponseField name="booking.hargaPaket" type="number">
  Total harga paket
</ResponseField>

<ResponseField name="booking.dpAmount" type="number">
  Jumlah DP yang harus dibayar
</ResponseField>

<ResponseField name="booking.payments" type="array">
  Riwayat pembayaran
</ResponseField>

<ResponseField name="summary" type="object">
  Ringkasan: `totalBayar`, `sisaTagihan`, `lunas`
</ResponseField>

<RequestExample>

```bash cURL
curl https://yourdomain.com/api/public/invoice/BK2604-A1B2
```

</RequestExample>
