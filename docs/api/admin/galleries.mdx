## Upload Photos

```bash
POST /api/admin/galleries/{id}/upload
```

Upload multiple files ke Cloudinary dan simpan ke database.

<ParamField body="files" type="file[]" required>
  Array file gambar (max 100 files, max 15MB per file)
</ParamField>

<ParamField body="accountId" type="string">
  Cloudinary account ID (opsional, default: account default)
</ParamField>

### Allowed Formats

- JPEG, PNG, WebP, HEIC/HEIF, TIFF, AVIF

### Upload Flow

<Steps>
  <Step title="Validasi">MIME type, file size, magic bytes verification</Step>
  <Step title="Upload">Batch upload ke Cloudinary (5 files per batch)</Step>
  <Step title="Simpan">`createMany` ke database</Step>
  <Step title="Rollback">Jika DB gagal, hapus file dari Cloudinary</Step>
</Steps>

## Photo Management

### List Photos

```bash
GET /api/admin/galleries/{id}/photos
```

### Delete Photo

```bash
DELETE /api/admin/galleries/{id}/photos/{photoId}
```

### Bulk Delete Photos

```bash
POST /api/admin/galleries/{id}/photos/bulk
Body: { "ids": ["photo-uuid-1", "photo-uuid-2"] }
```

## Selection Management

### List Selections

```bash
GET /api/admin/galleries/{id}/selections
```

Response termasuk stats: `total`, `edit`, `print`, `locked`.

### Toggle Lock

```bash
PATCH /api/admin/galleries/{id}/selections
Body: { "selectionId": "...", "isLocked": true }
```

### Delete Selection

```bash
DELETE /api/admin/galleries/{id}/selections
Body: { "selectionId": "..." }
```
