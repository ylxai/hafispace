## Get Dashboard Metrics

Ambil statistik dashboard untuk vendor yang sedang login. Response di-cache selama 5 menit.

### Response

<ResponseField name="overview" type="object">
  Ringkasan statistik
</ResponseField>

<ResponseField name="overview.totalBookings" type="number">
  Total booking sepanjang waktu
</ResponseField>

<ResponseField name="overview.bookingsBulanIni" type="number">
  Booking baru bulan ini
</ResponseField>

<ResponseField name="overview.totalClients" type="number">
  Total klien
</ResponseField>

<ResponseField name="overview.totalOmset" type="number">
  Total omset (semua pembayaran)
</ResponseField>

<ResponseField name="overview.pemasukanBulanIni" type="number">
  Pemasukan bulan ini
</ResponseField>

<ResponseField name="overview.dpBulanIni" type="number">
  DP diterima bulan ini
</ResponseField>

<ResponseField name="topPackages" type="array">
  Top 5 paket terpopuler
</ResponseField>

<ResponseField name="upcomingSessions" type="array">
  Sesi foto bulan ini (PENDING, CONFIRMED, COMPLETED)
</ResponseField>

<ResponseField name="recentBookings" type="array">
  5 booking terbaru
</ResponseField>

<ResponseField name="trendData" type="array">
  Tren pemasukan 6 bulan terakhir
</ResponseField>

<RequestExample>

```bash cURL
curl https://yourdomain.com/api/admin/metrics \
  -H "Cookie: next-auth.session-token=..."
```

</RequestExample>
