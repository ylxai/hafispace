Hafiportrait Platform adalah platform manajemen galeri foto profesional untuk fotografer pernikahan dan event. Dibangun dengan **Next.js 15** (App Router), **TypeScript** strict, dan **Prisma ORM**.

## Fitur Utama

## Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | Next.js 15, React 19, TypeScript 5, Tailwind CSS 4 |
| **Backend** | Next.js API Routes |
| **Database** | PostgreSQL (Neon) via Prisma ORM 5 |
| **Auth** | NextAuth.js 4 (JWT, Credentials) |
| **Storage** | Cloudinary (preview/thumbnail), Cloudflare R2 (planned) |
| **Realtime** | Ably |
| **Cache** | Upstash Redis |
| **Validation** | Zod |
| **Logging** | Pino (server-side) |
| **Monitoring** | Sentry |

## Data Models

Platform menggunakan 13 model Prisma dengan PostgreSQL:

- **Vendor** — Fotografer/studio, owner semua resource
- **Client** — Data klien, relasi ke booking
- **Booking** — Sesi foto (paket, harga, tanggal, lokasi)
- **Gallery** — Galeri foto dengan token akses unik
- **Photo** — Foto individual, disimpan di Cloudinary
- **PhotoSelection** — Seleksi foto oleh klien
- **Payment** — Record pembayaran
- **Package** — Paket fotografi
- **GallerySetting** — Konfigurasi per galeri
- **VendorCloudinary** — Multi-akun Cloudinary per vendor
- **CustomField** — Field custom untuk booking form
- **Notification** — Notifikasi in-app
- **ActivityLog** — Log aktivitas
